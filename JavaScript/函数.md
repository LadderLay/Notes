# 概念

## first-class objects  第一类对象

    函数与对象共存。函数也是对象，可以实现对象的所有功能，唯一区别在于函数是**可调用的**（invokable）。

- 回调函数 callback
第一类对象——函数能够作为另一个函数的参数传入，可以通过参数调用该函数 -> 回调函数

- 存储函数
函数属性——累加器

- 自记忆函数
函数属性——缓存

## 函数定义
1. 函数声明与函数表达式
- 函数声明(独立的)
    ```
    function myFun(){
        //code
    }
    ```
- 函数表达式
    ```
    var a = funtion(){...}
    ```
- 两者在代码中的位置不同。同时，对于函数声明来说，函数名 是强制的；对函数表达式来说，函数名是可选的。

- 立即函数
IIFE
